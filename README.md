# Multi-Step Form

This is a responsive multi-step form created with React.

## Setup Instructions

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/multi-step-form.git
   cd multi-step-form
2. npm install
3. npm start
4. Open your browser and navigate to http://localhost:3000.

 Project Structure
src/components/: Contains the form steps and navigation components.
src/App.js: The main App component that combines all steps.
src/styles.css: Basic styling and responsive design.

Features
Multi-step form with validation.
Local storage persistence.
Responsive design.
Tabbed navigation between steps

Optional Enhancements
Simulated API calls with error handling.
CSS transitions between steps.
Unit tests with Jest.

Assumptions and Decisions
Form data is persisted to local storage on each step change.
Client-side validation is handled with Formik and Yup.
Basic responsive design with CSS.
